# Utopian Score

This project can view your utopian contribution score based on questions and answers that moderator reviewed.

## Requirement
```
node.js
yarn
```

## Install
Clone this repo
```
git clone https://github.com/damaera/utopian-score
```

### development
```
yarn start

```
or 

```
yarn dev
```

### build for production
```
yarn build
```
